Review Prototype

BFS to USCG STCW firefighting cross-reference

This review page compares Maine Basic Fire School session completions against selected USCG NVIC firefighting competencies. It is intended to support discussion, validation, and gap review.

Draft for external review only. This prototype was prepared to support discussion between Oddly True AI and Maine Fire Service regarding possible alignment between Maine Basic Fire School and selected USCG STCW firefighting competencies. It is not an official equivalency determination, training record, certification tool, or operational guidance. All mappings and credit assumptions require review and validation by the appropriate authority before use.

How to use this page

  • Select the BFS sessions a candidate has completed.
  • Review the summary to see which USCG items appear fully covered, partially covered, or still unmet.
  • Use the detailed table to discuss any shipboard-specific competencies that still require separate evaluation.
