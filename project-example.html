Project Example: Build Something with AI (Step-by-Step)

You will build a small project step by step using AI. Follow each step in order.

What you will do

  • Pick one small project idea
  • Create a simple goal
  • Create a short project brief
  • Plan with AI in one chat
  • Draft in markdown
  • Improve the result
  • Review it yourself
  • Turn it into a final shareable result

How to use this page

Start with steps 1 to 4 first. That gives you your idea, goal, brief, and plan. Then move on to drafting and finishing.

1. Pick Your Project

Choose something simple that you already understand.

Good examples

  • Training guide
  • Checklist
  • Study sheet
  • Simple how-to

Pick something you already understand. AI works best when you bring the knowledge.

2. Create Your Project Goal

Start by creating a simple goal for your project.

Example

# Project Goal
Create a one-page guide that explains an idea, solves a small problem, or helps someone complete a task

Audience: Non-technical users
Tone: Simple, clear, friendly
Output: Markdown document

You can use this in your first planning chat.

Your turn

Modify this example so it matches your own project idea.

3. Create Your Project Brief

This keeps all your chats consistent.

What this is

# Project Brief
Goal: Create a simple document
Audience: Non-technical users
Tone: Clear, friendly
Output: Markdown
Rules:
- Keep it simple
- Avoid jargon
- Use short sections
- Ask questions if needed

What to do

Put this in your project source section first. Then reuse it in your chats.

4. Start Your Planning Chat

Now we use AI to think before we write.

Plan the project using both the goal and the brief.

Planning Prompt

Here is my project goal:

[paste your goal here]

Here is my project brief:

[paste your brief here]

Help me plan this project.
Ask me questions one at a time, then help me create a simple outline before writing.

Keep Your Chats Organized

Rename your chats as you go so you can come back to them later.

  • Plan
  • Draft
  • Revise
  • Review

5. Draft the Document

Now create the first version.

Draft Prompt

Using the outline we created, write a clean markdown document.

Include:
- title
- intro
- sections
- bullet points

Keep it simple and beginner-friendly.

If markdown looks unfamiliar, use the Markdown Quick Guide.

6. Improve the Result

Do not accept the first answer. Improve it.

Improve Prompt

Review this document and improve:
- clarity
- structure
- readability

Keep it concise.

7. Final Human Review

You are still responsible for the final result.

  • Is it clear?
  • Is it correct?
  • Is anything missing?
  • Would a beginner understand it?

8. Publish Your Work

Save your markdown and turn it into a final document.

Finishing stage

You already have the main content by this point. Now you are packaging it so someone else can use it.

Options

  • Create a Halo (quickest way to share) – halolumin.io (one simple sharing option)
  • Save as a document (Word, PDF, etc.)
  • Upload to a system you already use

The goal is to produce something real you can share.

9. Review Your Final Output

Now take your final result and run one more review pass.

Review Prompt

Here is my final document (or link):

[paste your content or link here]

Please review this and give me suggestions to improve:
- clarity
- usefulness
- organization
- readability for beginners

Tell me what is good, what is weak, and what I should change next.

10. Key Idea

You are not just asking AI for answers.

You are building something step by step:

  • Plan
  • Draft
  • Improve
  • Review
  • Publish
