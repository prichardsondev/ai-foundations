Session 3

Using Projects to Build with AI

This session is about moving from single prompts to building something with AI inside one project.

Session outcomes

  • Explain what a project is in simple terms
  • Understand why chats drift without structure
  • Use a short session brief to keep work aligned
  • See one strong planning prompt
  • Understand the high-level workflow from plan to publish

What this session covers

  • What a project is and why it helps
  • Why separate chats can drift away from your goal
  • How a short session brief keeps work consistent
  • One strong example and one strong planning prompt
  • The simple workflow from plan to publish

1. What is a Project?

A project is a workspace for one goal.

Simple definition

A project keeps your chats, instructions, and files together in one place.

Think of it like one workspace for one job. Instead of starting over each time, you keep everything connected.

2. The Problem: Chats Drift

Without structure, different chats can lose context and consistency.

One chat may sound clear and simple. Another may change the tone, miss the audience, or format the result differently.

That is why projects need a small amount of structure before you start.

3. The Solution: A Session Brief

A short set of instructions tells AI what you are building and how to help.

Example

A goal says what you want to make. A brief says how AI should help. A prompt is the message you type to start the work.

# Session Brief

Goal:
Create a one-page guide that explains an idea, solves a small problem, or helps someone complete a task

Audience:
Non-technical users

Tone:
Simple, clear, step-by-step

Output:
Markdown document

Rules:
- Keep it simple
- Use bullet points
- Avoid technical terms
- Ask questions if something is unclear

You write this once and reuse it across all chats.

4. Why This Works

A brief gives every chat the same starting point.

It keeps tone, structure, and output consistent across chats.

Instead of repeating yourself in every new conversation, you reuse the same brief and stay aligned.

5. Example Planning Prompt

Use the brief when you begin planning.

What to do right now

First, copy your session brief. Next, open a new chat. Then paste the brief and use the planning prompt below.

Here is my session brief:

[paste your brief here]

Help me plan this project.

Ask me questions one at a time, then help me create a clear outline before writing anything.

Keep the process simple and high-level.

  • Plan
  • Draft
  • Improve
