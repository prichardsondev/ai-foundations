Markdown Quick Guide

Markdown is a simple way to format text. You do not need to memorize it. AI can generate it for you, and you can review it before sharing.

What this page covers

  • Simple headings
  • Bullet points
  • Numbered lists
  • Bold text
  • Where markdown is useful

1. Headings

Use heading symbols to create titles and sections.

# Title
## Section
### Subsection

More # symbols means a smaller heading level.

2. Bullet Points

Use dashes to create a simple list.

- item one
- item two
- item three

3. Numbered Lists

Use numbers when order matters.

1. step one
2. step two
3. step three

4. Bold Text

Use double asterisks to make text bold.

**important text**

5. Combined Example

Here is a small example that uses several markdown features together.

# My Guide

## Steps
1. Do this
2. Do that

## Tips
- Keep it simple
- Review your work
- Share the final version

**Final reminder:** check your work before publishing.

6. Where Markdown Is Used

Markdown is most useful when the tool you use knows how to render it.

  • Documentation tools
  • Note-taking apps that support markdown
  • Web publishing tools
  • Halo Lumin and similar tools that turn markdown into a shareable page

If you send raw markdown in a normal message or email, it may not render. The other person may just see the symbols and formatting marks.

7. Helpful Reminder

You do not need to write markdown from scratch.

Simple tip

Ask AI to format your content as markdown. Then paste it into a tool that renders markdown, convert it to a document, or publish it as a shareable page.
