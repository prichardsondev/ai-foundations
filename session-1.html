1. What AI actually is

The core idea

AI tools like ChatGPT, Claude, or Grok are powerful prediction machines. They have been trained on enormous amounts of text and learned statistical patterns of language.

Two important stages

Training: The model studies massive datasets to learn patterns.
Inference: When you give it a prompt, it generates a response based on what it learned.

Simple analogy

Imagine a very well-read assistant who has seen millions of examples of good writing — but it doesn’t truly “understand” like a human. It just predicts what should come next.

2. What is a GPT?

Generative

It creates new text instead of just copying old answers.

Pre-trained

It learned from huge amounts of data before you started using it.

Transformer

It can understand the full conversation context, not just the last message.

What GPT tools are great at

  • Drafting emails, reports, and summaries
  • Brainstorming ideas
  • Explaining concepts clearly
  • Reorganizing or simplifying text

Important limitations

  • They can be confidently wrong (“hallucinations”)
  • They don’t know your specific workplace or recent events
  • They cannot truly verify facts

3. Safety check before prompting

Before you paste anything into a public AI tool, pause for a quick privacy and verification check.

Use safe examples today

Use fictional, public, or sanitized examples. Do not paste sensitive workplace information, private customer data, employee records, financial details, passwords, confidential documents, or anything you would not be allowed to share outside your organization.

Plain-English safety habit

  • Replace real names with roles like “customer,” “vendor,” or “supervisor”
  • Remove account numbers, addresses, phone numbers, and internal identifiers
  • Use made-up numbers when the exact numbers are not needed
  • Check important answers against a trusted source before acting on them

4. Better Prompting – The 5-Part Framework

A well-structured prompt dramatically improves the quality of the AI’s response.

The 5 parts of a strong prompt

  1. Role – Tell the AI who it should act as
  2. Task – Clearly state what you want done
  3. Context – Provide necessary background
  4. Constraints – Add limits on tone, length, or style
  5. Output Format – Specify how the answer should be presented

5. Build Your AI Starter Kit

Your Prompt Library is the saved file where your reusable prompts live. Your AI Starter Kit is the full workflow: AI Working Agreement, Personal Context Prompt, Task Playbook Prompt, and one test chat or project.

Activity goal

If this were the only AI class you ever took, this is the workflow I would want you to leave with.

Use fictional, public, or sanitized examples while you build. Save each result in one Prompt Library file so you can reuse it later.

Step 1

Create your AI Working Agreement

This prompt tells the AI how you want to be helped. It can set expectations for plain language, useful questions, honest uncertainty, and reminders to verify important information.

Copy/paste builder prompt

Help me create my AI Working Agreement.

Ask me 5 short questions about how I want AI to help me.

Use my answers to draft a short reusable AI Working Agreement that includes:
- How I want responses written
- When the AI should ask clarifying questions
- How the AI should handle uncertainty
- How the AI should remind me to verify important information
- What kinds of information I should not paste into public AI tools

Keep the language plain-English and practical.

Save the result in a Halo, document, or notes file under My AI Working Agreement.

Step 2

Create your Personal Context Prompt

This gives the AI safe background about your role, goals, preferences, and common work. Keep it sanitized and non-sensitive. Do not include private, confidential, customer, employee, financial, legal, medical, or internal company information.

Copy/paste builder prompt

Help me create a safe Personal Context Prompt.

Ask me 6 short questions about:
- My general role or type of work
- The kinds of tasks I want help with
- My preferred writing style
- My audience
- My skill level with AI
- Any safe boundaries or preferences

Do not ask for confidential, private, customer, employee, financial, legal, medical, or internal company information.

After I answer, draft a reusable context prompt I can paste at the start of future AI chats.

Save the result under the same My Prompt Library file, below your AI Working Agreement.

Step 3

Create one Task Playbook Prompt

Task Playbook Prompts are reusable prompts for common work tasks. Pick one task you do often, then let AI interview you and turn it into a reusable template.

Task ideas to choose from

  • Email helper
  • Meeting summary helper
  • SOP builder
  • Presentation builder
  • Spreadsheet / number analysis helper
  • Report or memo helper
  • Training material builder
  • Troubleshooting helper
  • Research / comparison helper

Copy/paste builder prompt

Help me create a reusable Task Playbook Prompt.

First, interview me with short questions about the repeatable task I want help with.

Ask about:
- The task type
- The audience
- What information I can safely provide
- What output I need
- Tone, length, or format preferences
- How I should review the result before using it

Do not ask me to provide sensitive workplace information.

After I answer, create a reusable task prompt template using this structure:
Role:
Task:
Context needed:
Constraints:
Output format:
Review checklist:

Include blanks I can fill in next time.

Save the finished template under Task Playbook Prompts in your Prompt Library.

Step 4

Use your three prompts together

Start a new ChatGPT chat or project. Paste your three saved prompts in this order:

  1. My AI Working Agreement
  2. My Personal Context Prompt
  3. One Task Playbook Prompt

Then test the setup with a fictional, public, or sanitized task. This is the beginning of your reusable AI workspace.

Sanitized test task ideas

  • Draft a professional follow-up email from fictional notes
  • Turn messy meeting notes into action items
  • Create a 5-slide presentation outline
  • Summarize a public article
  • Build a simple checklist from a fake SOP

Simple test instruction

“Use the prompts above to help me with this sanitized task. Ask any missing questions first, then draft the output and include a short review checklist.”

6. Quick Tips for Better Results

  • Be as specific as possible
  • Give examples when it helps
  • Ask the AI to break complex tasks into clear steps
  • Iterate — improve your prompt based on the first response
  • Always review important outputs carefully

7. Closing activity: save and test

By the end of Session 1, you should have a saved Prompt Library and one test chat or project using your three prompts.

What to leave with

  • A Halo, document, or notes file called My Prompt Library
  • Your AI Working Agreement saved in that file
  • Your Personal Context Prompt saved in that file
  • One Task Playbook Prompt saved in that file
  • One test chat or project where you tried all three prompts together

Bring this file back in future sessions so you can improve it as your AI skills grow.
